# Questão 1
## Estratégia Dinâmica de Algoritmos de Risco (Strategy Pattern)
Um sistema financeiro precisa calcular métricas de risco (Value at Risk, Expected Shortfall e Stress Testing) que podem mudar dinamicamente durante a execução.

# Questão 2
## Integração com Sistema Bancário Legado (Adapter Pattern)
O sistema moderno de processamento de transações usa uma interface limpa, enquanto o sistema bancário legado usa HashMap, codificação antiga de moedas e campos obrigatórios inconsistentes.

# Questão 3
## Controle de Estados de Usina Nuclear (State Pattern + Rules Engine)
Uma usina nuclear opera em estados como OPERACAO_NORMAL, ALERTA_AMARELO, ALERTA_VERMELHO e EMERGENCIA. Cada transição exige validações complexas e algumas são irreversíveis.

# Questão 4
## Cadeia de Validadores de NF-e (Chain of Responsibility + Circuit Breaker + Rollback)
Uma NF-e precisa ser validada por múltiplas regras encadeadas, como XML, certificado digital, regras fiscais, banco de dados e SEFAZ, com ordem condicional e rollback.
